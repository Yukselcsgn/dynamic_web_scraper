### 1.2 Kategori Listesi Çekme
**Amaç:** Bir kategorideki tüm ürünleri listelemek

**Senaryo:**
- Kullanıcı kategori URL'i verir (örn: laptop kategorisi)
- İlk 50 ürünün bilgilerini çeker
- Pagination yok, sadece tek sayfa

**Özellikler:**
- Liste extraction
- Temel CSS selector kullanımı
- JSON ve CSV export
