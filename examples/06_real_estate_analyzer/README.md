### 2.3 Emlak Fiyat Analizi (Sahibinden, Emlakjet vb.)
**Amaç:** Belirli bir bölgedeki emlak fiyatlarını analiz etmek

**Senaryo:**
- Kullanıcı şehir ve ilçe seçer
- Tüm ilanları çeker (kira/satılık)
- m² başına fiyat analizi yapar
- Mahalle bazlı karşılaştırma grafiği oluşturur

**Özellikler:**
- Multi-page scraping
- Data cleaning & normalization
- Statistical analysis
- Interactive charts (Plotly)
