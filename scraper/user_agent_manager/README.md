# User Agent Manager Package

The `user_agent_manager` package provides user agent management and rotation.

## Modules

### 1. User Agent Manager (`user_agent_manager.py`)
Main user agent management interface.

### 2. User Agent Loader (`user_agent_loader.py`)
Loads user agent strings from various sources.

## Features

- Realistic user agent strings
- Browser diversity (Chrome, Firefox, Safari, Edge)
- Platform diversity (Windows, Mac, Linux)
- Automatic rotation
- Up-to-date user agent database

## Usage

The user agent manager helps avoid detection by rotating through realistic browser user agent strings.
