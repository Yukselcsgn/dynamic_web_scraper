# Site Detection Package

The `site_detection` package provides intelligent website type and structure detection.

## Modules

### 1. Site Detector (`site_detector.py`)
Detects website type (e-commerce, blog, news, etc.).

### 2. Smart Detector (`smart_detector.py`)
Advanced pattern-based site detection with confidence scoring.

### 3. HTML Analyzer (`html_analyzer.py`)
Analyzes HTML structure for patterns and features.

### 4. CSS Selector Builder (`css_selector_builder.py`)
Builds optimal CSS selectors based on site structure.

### 5. Site Config (`site_config.py`)
Site-specific configuration management.

## Features

- Automatic site type detection
- Pattern recognition
- Confidence scoring
- Adaptive selector generation
- Site-specific optimizations

## Usage

The site detection system automatically identifies website patterns and generates appropriate selectors, reducing configuration overhead.
