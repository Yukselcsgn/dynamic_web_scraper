# Data Parsers Package

The `data_parsers` package provides utilities for parsing and processing scraped data.

## Modules

### 1. Data Parser (`data_parser.py`)
Core data processing and saving functionality.

**Key Functions:**
- `save_data()`: Save data in CSV or JSON format
- `process_data()`: Clean and normalize raw data

### 2. HTML Parser (`html_parser.py`)
HTML-specific parsing utilities.

### 3. JSON Parser (`json_parser.py`)
JSON-specific parsing utilities.

## Usage

These parsers provide common data processing operations for cleaning, normalizing, and saving scraped data in various formats.
