# Utils Package

The `utils` package provides common utility functions used throughout the scraper.

## Modules

### 1. HTML Utils (`html_utils.py`)
HTML processing utilities.

### 2. Parse Utils (`parse_utils.py`)
Data parsing and extraction utilities.

### 3. Proxy Utils (`proxy_utils.py`)
Proxy-related helper functions.

### 4. Request Utils (`request_utils.py`)
HTTP request utilities and helpers.

### 5. User Agent Utils (`user_agent_utils.py`)
User agent related utilities.

### 6. Wait Utils (`wait_utils.py`)
Timing and delay utilities.

## Usage

These utility modules provide common functionality shared across different scraper components, promoting code reuse and consistency.
