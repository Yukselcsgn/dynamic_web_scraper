# Plugins Package

The `plugins` package provides a plugin system for extending scraper functionality.

## Modules

### Plugin Manager (`plugin_manager.py`)
Dynamic plugin loading and management system.

**Key Features:**
- Plugin discovery and loading
- Hook-based architecture
- Pre/post-fetch hooks
- Data transformation plugins
- Custom extractor plugins
- Plugin configuration
- Plugin lifecycle management

## Usage

The plugin system allows users to extend the scraper with custom functionality without modifying core code.
