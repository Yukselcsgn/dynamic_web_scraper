# Proxy Manager Package

The `proxy_manager` package provides proxy management and rotation functionality.

## Modules

### 1. Proxy Manager (`proxy_manager.py`)
Main proxy management interface.

### 2. Proxy Loader (`proxy_loader.py`)
Loads proxies from various sources.

### 3. Proxy Validator (`proxy_validator.py`)
Validates proxy functionality.

### 4. Proxy Rotator (`proxy_rotator.py`)
Implements proxy rotation logic.

## Features

- Multiple proxy source support
- Proxy validation and testing
- Automatic rotation
- Dead proxy detection
- Performance tracking

## Usage

The proxy manager helps avoid IP blocking by rotating through multiple proxy servers during scraping operations.
